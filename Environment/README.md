# Environment vs Prop Drilling (SwiftUI)

This project demonstrates how SwiftUI shares state using @Environment compared to traditional prop drilling.

## Concepts Covered
- Prop Drilling problem
- Shared state architecture
- @Environment injection system
- Observable theme management

## Key Insight
Create → Inject → Read

- PropDrilling/ → Traditional approach
- Environment/ → SwiftUI approach
