# Generics (Swift)

This project explores why Generics exist by first building duplicate implementations, then replacing them with a single reusable solution.

## Concepts Covered

- Concrete types
- Code duplication
- Generic types (`T`)
- Type-safe abstraction

## Key Insight

Same Logic → Different Types → Generics
